A modern full-stack web application built with FastAPI that allows users to order and store details in an Excel file.

Food Store is a lightweight, interactive web application where users can:

> Enter their name

> Select a food item

> Select the quantity

> Place Order/ Cancel Order using clear

> The application uses a FastAPI backend and stores user data in an Excel file using Pandas.

> It also includes a poster display based on the most number of food items ordered.
